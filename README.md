# spring-cloud-kafka-saga-pattern

Pre-requisites:
Install Kafka
Install Spring
Install Java

Start Zookeper 
--------------

zookeeper-server-start.bat d:\Software\kafka_2.13-2.5.0\config\zookeeper.properties


Start Kafka server
------------------

kafka-server-start.bat d:\Software\kafka_2.13-2.5.0\config\server.properties
